When binding data to selections, there will be either more, less, or the same number of DOM elements as there are data values. When you have more data values than DOM elements in the selection, you trigger the .enter() function, which allows you to define behavior to perform for every value that doesnâ€™t have a corresponding DOM element in the selection. Let's look at an example

This example is very similar to what we've been using to this point.

    
    <div id="guitars"></div>

    <script>
        d3.select('#guitars')
            .selectAll('.guitar')
            .data(['Fender', 'Gretch', 'Ibanez', 'Lowden', 'Taylor'])
            .enter()
            .append('div')
            .attr('style', 'border: 1px solid pink;')
            .text(d => d)
    </script>

Let's step through what's going on here:

  • We begin by making an empty selection
  • We then bind the data to our selection
  • We then use enter() to determine how to respond if there are more data elements than there are elements in the selection.
  • We then append a <div> for each item in the selection
