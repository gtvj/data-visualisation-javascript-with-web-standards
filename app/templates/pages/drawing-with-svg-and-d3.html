Drawing with SVG and D3

To work with DOM elements using D3 we need to turn them into D3 selection elements.

D3 offers a fluent API allowing for calls to be chained.

The code

    
    const svg = d3.select('.visualisation').append('svg').attr('width', 500).attr('height', 300);

    const stick = svg.append('g').attr('transform', 'translate(100,100)');

    stick.append('line')
        .attr('x2', 300)
        .style('stroke', 'black');

    stick.append('circle')
        .attr('cx', 300)
        .attr('r', 3);

    stick.append('text')
        .attr('y', -10)
        .text('This is a line')
