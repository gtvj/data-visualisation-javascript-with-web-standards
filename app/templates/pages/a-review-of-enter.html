A review of core D3 functions

Selection and binding

At its core, D3 is about selecting and binding.

Selections are groups of one or more web page elements that may be associated with a set of data D3.js in Action, Second edition

Here's an example selection

    
    d3.selectAll('.guitar') // A selection
    d3.selectAll('.guitar').data(['D18','J185','Lowden']) // A selection with bound data

The process of associating data to page elements is known as data binding and the D3 visualisation process is about using data to set the appearance of bound elements.
