SVG basics - using defaults

No D3 here, again.

In SVG, '0' is the default for attributes. Because of this you can omit them and delegate some positioning to the group element.

This example demonstrates using this identical shapes with different translations applied to the parent group.

The code

    
    <svg width="500" height="300">
        <g transform="translate(100,200)">
            <line x2="300" stroke="black"></line>
            <circle r="3" cx="300"></circle>
            <text y="-10">Here's another line</text>
        </g>
        <g transform="translate(100,100)">
            <line x2="300" stroke="black"></line>
            <circle r="3" cx="300"></circle>
            <text y="-10">Here's a line</text>
        </g>
    </svg>
